README

工具功能如下

  1. 自动导出你的backdoor和mof文件，

  2. 自动判断mysql版本，根据版本不同导出UDF的DLL到不同目录，UDF提权

  3. 导出LPK.dll文件，劫持系统目录提权

  4. 写启动项提权

工具仅做方便使用，技术含量几乎没有。

用py脚本写的，也打包成exe了，个人用的还算顺手。
